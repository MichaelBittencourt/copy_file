This repository was created to explain the GNU make and Makefile


## Clone Project

```bash
$ git clone https://github.com/MichaelBittencourt/copy_file.git
$ cd copy_file
```

## Running

```bash
$ make
$ ./hello
```

## Cleaning

```bash
$ make clean
```

## Create debug binary

This option is available to version 2 and 3

```bash
$ make DEFINES=-DDEBUG
```

## Create assembly files

This option is available to version 2 and 3

```bash
$ make assembly
```
